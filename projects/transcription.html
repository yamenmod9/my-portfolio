Audio Transcription

AI-powered speech-to-text conversion

About This Project

An advanced speech recognition system that converts audio recordings into accurate text transcripts with speaker identification and punctuation.

Date: November 2022

Python TensorFlow Web Audio API Flask

Key Features

  • Real-time audio transcription
  • Speaker diarization (who spoke when)
  • Automatic punctuation
  • Export to multiple formats (TXT, DOCX, SRT)
  • Custom vocabulary support
